# 日报检查清单

> 发布前逐项核对，全部通过方可发布。

## 格式检查

- [ ] 每条标题使用 `[[来源] 标题](URL)` 格式，来源和标题一体放入超链接
- [ ] 链接指向具体文章页面，非网站首页
- [ ] 标题下方不重复显示裸链接
- [ ] 标题下方有 1-2 句内容摘要

## 板块检查

- [ ] 🔥 今日必读：2-4 条最重要的新闻
- [ ] 🤖 AI 赛道速览：AI 行业动态（模型、产品、融资、开源）
- [ ] 🛒 消费行业 AI 落地：零售/餐饮/电商/品牌 AI 应用
- [ ] 🎬 达人视点：B站/YouTube 达人评测、观点
- [ ] 🔬 深度值得关注：长文、报告、论文

## 内容检查

- [ ] 所有链接可访问（非 404）
- [ ] 数据准确，有来源支撑
- [ ] 日期正确（聚焦昨天的内容）
- [ ] 无重复条目
- [ ] 来源覆盖 ≥ 5 个
- [ ] 总条目 ≥ 15 条

## 质量检查

- [ ] 摘要信息量充足，读者可判断是否值得点开
- [ ] 无错别字
- [ ] 无 AI 幻觉内容（无法核实的标注 [Unverified]）
- [ ] 板块标题使用正确的 emoji

## 发布检查

- [ ] 已通过 message 工具发送到飞书
- [ ] 已保存 Markdown 到 daily/YYYY-MM-DD-ai-consumer.md

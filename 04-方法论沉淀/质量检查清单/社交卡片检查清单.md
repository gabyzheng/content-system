# 社交卡片检查清单

> 发布前逐项核对，全部通过方可发布。

## 设计检查

- [ ] 模板：Swiss International（非 Editorial）
- [ ] 配色：IKB 蓝（#002FA7），整组统一
- [ ] 尺寸：1080×1440（3:4）
- [ ] 无圆角（border-radius: 0）
- [ ] 无阴影（box-shadow: none）
- [ ] 无渐变（no linear-gradient）
- [ ] 字体：Inter + Noto Sans SC，无衬线

## 排版检查

- [ ] "the larger, the lighter" — 大字用轻字重（200-300），小字用重字重（500-600）
- [ ] 安全边距：左右 80-96px，上 72-112px，下 80-120px
- [ ] 最小可读字号 ≥ 22px（1080×1440 画板）
- [ ] 中文行高 1.35-1.55，标题行高 1.02-1.08

## 内容检查

- [ ] 每张卡片一个核心信息
- [ ] 数据准确，与源内容一致
- [ ] 标题简洁（≤ 12 字）
- [ ] 无错别字

## 渲染检查

- [ ] Playwright 渲染成功，无报错
- [ ] 输出 PNG 尺寸 = 1080×1440
- [ ] 字体加载正常（非回退字体）
- [ ] 所有 Lucide 图标正常渲染

## 发布检查

- [ ] 已通过 message 工具发送 PNG
- [ ] 已保存 HTML 到 04-已发布内容/社交卡片/

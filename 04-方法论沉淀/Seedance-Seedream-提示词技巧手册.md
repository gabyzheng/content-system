## 一、Seedance 2.0（视频生成）

### 1.1 万能公式（八个核心要素）

```
精准主体 + 动作细节 + 场景环境 + 光影色调 + 镜头运镜 + 视觉风格 + 画质参数 + 约束条件
```

先锁定"谁"和"在干什么"，再交代"在哪、什么氛围"，然后告诉模型"怎么拍"，最后用风格、画质和约束收紧结果。

### 1.2 分镜时序（最重要技巧）

按时间线组织成"镜头1 / 镜头2 / 镜头3"，不要一口气写一大段无结构的剧情。

**反面**：男人在街头紧张地奔跑，画面很有电影感。

**正面（15秒追逐戏）**：
- 镜头1：街巷侧拍，男人缓慢起跑，带有急促的呼吸感
- 镜头2：男人撞翻水果摊，镜头快速摇动并给到男人惊恐的特写
- 镜头3：男人翻过矮墙消失，镜头缓慢拉远定格在空荡的街道

### 1.3 动作写法

1. **优先写缓慢、连续的小动作**（缓慢行走、轻轻抬手、微微低头、顺势坐下），避免"狂奔、大跳、剧烈翻滚"
2. **写出动作间的过渡**（借着刚才转身的惯性、从停顿自然过渡到举手）
3. **情绪外化**——用身体信号替代抽象情绪词：

| 抽象情绪 | 外化为动作与细节 |
|---------|----------------|
| 悲伤 | 低头、肩膀微微颤抖、眼眶泛红、手指无意识地攥紧衣角、泪水在眼眶里打转但没有落下 |
| 喜悦 | 嘴角抑制不住地上扬、眉眼舒展、脚步变得轻快、下意识地哼起小曲、忍不住原地转个圈 |
| 紧张/焦虑 | 频繁地看手表、手指不停敲击桌面、呼吸急促、眼神闪躲、无意识地啃咬指甲 |
| 愤怒 | 双拳紧握、下颌线紧绷、胸口剧烈起伏、眼神如刀般锐利、从牙缝里挤出话语 |
| 释然 | 长长地舒了一口气、紧绷的肩膀完全放松下来、脸上露出久违的淡淡的微笑、抬头望向远方 |

### 1.4 运镜规则

- 一个镜头里**只指定 1 种运镜方式**，不要同时要求推拉摇移
- 中文运镜词理解力强：中景、特写、全景、缓慢推镜、平稳横移、固定镜头

### 1.5 约束词（必加）

```
面部稳定不变形、五官清晰、人体结构正常、动作自然流畅、不僵硬、画面无卡顿、无闪烁
```

### 1.6 画质与风格词（句末收紧）

```
高清，细节丰富，电影质感，色彩自然，光影柔和
赛博朋克冷蓝紫色调、复古胶片、日系清新
```

### 1.7 多参考能力（@ 语法）

**素材角色划分**（稳妥配置 4-5 个素材）：
- 角色图 1-2 张（正脸/全身）
- 场景图 1 张
- 运镜视频 1 段
- 音频 1 段

**基础写法**：
```
@图片1 作为角色参考，@图片2 作为场景参考，@视频1 参考运镜和动作节奏，@音频1 作为背景音乐
```

**进阶写法（绑定分镜时序）**：
```
@图片1 为角色外观。镜头1：角色背对镜头站立，镜头缓慢环绕；镜头2：角色转身，参考 @视频1 的运镜方式；镜头3：镜头快速拉远展现全景，配合 @音频1 的鼓点
```

**关键避坑**：
- 多人物/多视频时必须明确对应关系
- 例：`@图片1 的女生为女主，@图片2 的男生为男二，@视频1 仅用于学习镜头运动轨迹，不要复刻其中的人物和场景`
- 九宫格参考图效果差，应拆分成单张上传

### 1.8 角色一致性

- 每次生成都引用同一张角色参考图
- 提示词中加上：`基于 @图片1 保持角色外观、服装、发型一致`

### 1.9 视频延长

**推荐句式**：
```
将 @视频1 向后延长 10 秒，保持角色、场景、光影风格与前段完全一致
向前延长 @视频1，生成...
```

**策略选择**：
- 延长 → 文戏（长对话、情绪积累、单一路径移动），追求沉浸感
- 拼接 → 武戏（追逐、打斗、快速蒙太奇），追求节奏感

### 1.10 局部修改

绑定时间轴与空间层：
```
基于 @视频1 进行编辑。在 0-10 秒的整个时间轴内，将画面中主角身上的蓝色外套替换为红色外套。保持人物的面部、发型、动作以及背景环境、光线完全不变。
```

### 1.11 首尾帧控制

- 首帧约束：`@图片1 作为首帧` → 确保开场画面与参考图对齐
- 尾帧约束：`@图片2 作为尾帧` → 镜头落点精准收在指定画面

### 1.12 文字生成

**Slogan 公式**：
```
「文字内容」+「出现时机」+「出现位置」+「出现方式」，「文字特征（颜色、风格）」
```

**字幕**：
```
画面底部出现字幕，字幕内容为"......"，字幕需与音频节奏完全同步
```

**气泡台词**：
```
「角色」说："......"，角色说话时周围出现气泡，气泡里写着台词
```

### 1.13 三段论模板

```
整体设定 + 分镜时序 + 风格画质约束
```

**完整示例（都市情感）**：
```
@图片1 中的女孩作为主角，@图片2 作为宿舍场景风格参考，参考 @视频1 的运镜方式。

镜头1：傍晚时分，女孩脚步轻快地走到宿舍门口，镜头中景平稳跟拍，暖黄色日光从窗外洒进走廊，她在门口停顿一下，深呼吸，表情略带紧张。

镜头2：她推开门走进宿舍，镜头切到室内中景，舍友们一边整理书本一边抬头看向她，其中一人笑着问"考得怎么样呀，过了吗？"，镜头在几人之间缓慢切换半身特写。

镜头3：女孩先低头露出落寞表情，镜头给到她的近景，随后她抬头憋不住笑意，哈哈大笑说"骗你们的"，舍友们追着打闹起来，镜头缓慢拉远，定格在宿舍内一片欢声笑语的全景画面。

全程画面高清电影纪实风，色调温暖，光影柔和；人物面部稳定不变形，动作自然流畅，无卡顿无闪烁。
```

### 1.14 镜头语言词库

| 术语 | 定义 | 情绪指向 | 示例片段 |
|------|------|---------|---------|
| 大特写 (ECU) | 只展现面部局部（眼睛、嘴唇） | 强调微表情，放大极致情绪 | `...眼部大特写，瞳孔因惊恐而放大` |
| 特写 (CU) | 头到肩 | 聚焦面部表情，传递强烈情绪 | `...给到主角一个悲伤的特写，眼泪正从脸颊滑落` |
| 近景 (MCU) | 胸部以上 | 看清表情+兼顾肢体语言 | `...近景展现他无奈地摊开双手` |
| 中景 (MS) | 膝盖或腰部以上 | 对话场景，展现互动关系 | `...两人在吧台前的中景对话` |
| 全景 (FS) | 人物全身+小范围环境 | 交代完整动态和环境互动 | `...全景镜头下，她独自站在空旷的站台上` |
| 远景 (LS) | 人物占比小，环境为主体 | 强调氛围，表现渺小/孤独/宏大 | `...远景中，他的身影逐渐消失在茫茫雪原` |

### 1.15 轨道补齐

```
视频1 + 过渡画面描述 + 接视频2 + 过渡画面描述 + 接视频3
```
最多支持 3 段视频输入，总时长不超过 15s。

## 二、Seedream（图像生成）

### 2.1 通用提示词公式（5.0/4.5/4.0）

```
连贯自然语言描述画面内容（主体+行为+环境）+ 短词语描述画面美学（风格、色彩、光影、构图）
```

**示例**：
- ❌ 一个女孩，撑伞，林荫街道，油画般的细腻笔触使画面生动美好
- ✅ 一个女孩穿着华丽的服装，撑着遮阳伞走在林荫道上，莫奈油画风格

### 2.2 核心技巧

1. **专业词汇用英文/词源语言**，效果更准确
2. **写出图像用途和类型**：`PPT封面背景图`、`广告海报设计`、`纪实摄影`
3. **文字生成**：把文字内容插入 `""` 引号中 → `生成一张海报，标题为"Seedream V4.0"`
4. **组图生成**：用"一系列""组图""帮我生成几张图"触发，最多 9 张
5. **短提示词也能惊艳**：在表达准确的基础上，简单直接比堆砌形容词更有效

### 2.3 5.0 尺寸限制

- 最低 3,686,400 像素
- 必须用 `2K`（2848×1600）或更大
- 不能用 `1792x1024`（会返回 400 错误）

### 2.4 编辑能力（4.0）

**元素增删**：
```
编辑指令：给「主体」增加「元素」
编辑指令：去除「元素」
```

**改变风格**：
```
编辑指令：将画面改为「风格名称」风格
```

**改变光影色调**：
```
编辑指令：将画面改为「光影描述」
```

**人像编辑**：换装、换发型、换表情、改年龄、改姿势

**框选/涂抹编辑**：
```
编辑指令：删除框中的文字
编辑指令：将绿框中的字体改为可爱体
编辑指令：在图1蓝色涂抹区域增加图2的斜挎包
```

### 2.5 图表可视化

```
三维柱状图，色彩鲜明，展示 2023-2025 年全球三大新能源汽车品牌销量数据，横轴为年份，纵轴为销量（万辆），带数据标签，背景简洁
```

```
折线图，蓝色平滑曲线，呈现北京近3个月平均气温变化，横轴标月份，纵轴标温度（℃），网格线细腻
```

```
饼图，分4个色块，显示年度预算占比，A部门20%，B部门40%，C部门10%，D部门30%，图例清晰，整体风格商务简约
```

### 2.6 2.1/2.0/1.4 版本提示词结构

```
风格词 + 构图词 + 内容词（角色 + 在干什么/环境氛围）
```

**示例**：
```
超精细，masterpiece，highres，Anime style，厚涂风格，电影般的意境角度，细致的脸，一位白色衣服女孩，有着蓝色头发，悬浮在水下正在看书
```

### 2.7 减少画面中文字的方法

1. 关闭 `use_rephraser` 扩写字段（用户 prompt 详细时）
2. prompt 中避免出现"对话、说话、讨论、交谈、说、询问"
3. 负向提示词中添加：`text, logo, signature, username, watermark`

## 三、配图场景速查

### 公众号封面/插图
- 风格：Swiss 极简、editorial quality、lots of negative space
- 尺寸：5.0 用 `2K`（2848×1600），16:9
- 提示词结构：主体隐喻 + 构图分割 + 色调 + 风格词
- 约束：no text, no faces（如需后加文字）

### PPT 封面背景
- 用途词：`PPT封面背景图`
- 风格：蓝色简约咨询风、留白、插图服务于观点
- 尺寸：16:9

### 数据可视化
- 直接描述图表类型+数据+配色+风格
- 柱状图/折线图/饼图/地图热力

### 社交卡片
- 用途词：`社交媒体卡片`、`信息图`
- 风格：现代简约、品牌色系

### 短剧/视频配图
- 参考 Seedance 分镜方法论
- 先出关键帧图（Seedream），再生成视频（Seedance）

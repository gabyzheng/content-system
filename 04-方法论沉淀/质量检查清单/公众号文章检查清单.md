# 公众号文章检查清单

> 发布前逐项核对，全部通过方可发布。

## 内容检查

- [ ] 标题 ≤ 32 字（微信限制）
- [ ] 作者 ≤ 16 字
- [ ] 摘要 ≤ 128 字，信息量充足
- [ ] 正文 < 2 万字符（微信限制）
- [ ] 无错别字
- [ ] 无 AI 幻觉内容（无法核实的标注 [Unverified]）
- [ ] 数据有来源标注
- [ ] 原文链接已附（如有）

## 去 AI 化检查（硬约束）

- [ ] 已通过 humanizer 技能处理全文
- [ ] 无「pivotal」「crucial」「a testament to」式膨胀表达
- [ ] 无「——」破折号堆砌
- [ ] 无「好消息是」「更可怕的是」「值得注意的是」等替读者做情绪判断的短语
- [ ] 有「我」的感受，不是中性播报
- [ ] 句子长短交错，不整段同样节奏
- [ ] 具体 > 抽象，数据 > 形容词

## 排版检查

- [ ] 已按 design.md 规范生成 HTML（677px 宽）
- [ ] 主题已与用户确认（非擅自选择）
- [ ] 标题层级合理（H2/H3 递进）
- [ ] 表格样式正确
- [ ] 高亮块/引用块样式正确

## 配图检查

- [ ] 封面图已生成（900×383px，_cover.html）
- [ ] 封面文字与背景有足够对比度
- [ ] 正文插图位置正确（对应文章核心观点）
- [ ] 图片已 base64 内嵌或使用公网 URL
- [ ] 总 HTML payload ≤ 50KB（避免 413 错误）
- [ ] 插图风格统一（同主题、同色系）

## 社交卡片检查（可选，需明确指令触发）

- [ ] 已按 Guizang Swiss 规范生成卡片
- [ ] 卡片尺寸 1080×1440（3:4）
- [ ] 配色 IKB 蓝，无圆角无阴影无渐变
- [ ] 已通过 validate-social-deck.mjs 自动核查

## 推送检查

- [ ] 已 cd 到 weixin-mp-push 目录
- [ ] 已执行 `node push-to-wechat-mp.js wx6d5ea01d0b1e7156 html {文件名}.html`
- [ ] 推送成功（statusCode 200）
- [ ] 已告知用户去草稿箱确认

- [ ] 配图已归档

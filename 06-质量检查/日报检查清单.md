# 日报检查清单

## 格式检查
- [ ] 每条新闻使用 `<a href="URL">[来源] 标题</a>` 格式（飞书可点击链接）
- [ ] 无裸 URL 出现在正文中
- [ ] 每条新闻下方有 1-2 句摘要
- [ ] 无纯加粗文字代替链接
- [ ] 无 Markdown 链接格式 `[文本](URL)`（飞书不渲染）

## 内容检查
- [ ] 聚焦昨日（2026-06-19）数据
- [ ] 板块完整：🔥 今日必读 / 🤖 AI 赛道速览 / 🛒 消费行业 AI 落地 / 🎬 达人视点 / 🔬 深度值得关注
- [ ] 每条新闻有具体文章链接

## 发送检查
- [ ] message 和 text 参数内容完全一致（非占位符如 "dummy"）
- [ ] 消息完整发送（不只发标题）
- [ ] 超过长度限制时正确拆分
- [ ] 每条消息包含日报标题头

# H5-Deck（飞书风格 HTML 幻灯片）

使用 `feishu-deck-h5` 技能生成飞书风格暗色商务 HTML 幻灯片。

## 与 PPT 分支的区别

| | PPT 分支 | H5-Deck 分支 |
|---|---|---|
| 输出格式 | JSON → 图片模型出图 | 单文件 HTML |
| 交付物 | 逐页 PNG/PPTX | index.html |
| 交互 | 静态图片 | PC 全屏演示 + 移动端竖屏 |
| 编辑 | 需重新生成 | texts.md 文本侧车 + overrides.css |
| 适用 | 正式提案、打印、邮件附件 | 在线分享、嵌入网页、快速出片 |

# 竞品分析检查清单

> 发布前逐项核对，全部通过方可发布。

## 格式检查

- [ ] 使用表格进行对比分析
- [ ] 每条对比有明确的维度名称
- [ ] 结论有明确的推荐/建议

## 数据检查

- [ ] 所有数据有来源（官方文档、GitHub、实测）
- [ ] 版本号准确
- [ ] 价格/计费信息已核实
- [ ] 未核实内容标注置信度：[Verified] / [Inference] / [Unverified]

## 对标检查

- [ ] 对标维度覆盖 ≥ 10 个
- [ ] 每个维度有双方的具体表现描述
- [ ] 差异点有分析，非简单罗列
- [ ] 优势/劣势有数据支撑

## 质量检查

- [ ] 无错别字
- [ ] 无 AI 幻觉内容
- [ ] 结论可操作（不是"各有优劣"的废话）
- [ ] 标注了分析日期和版本

## 发布检查

- [ ] 已保存到 04-已发布内容/竞品分析/
- [ ] 核心发现已沉淀到 03-素材库/

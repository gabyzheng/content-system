### 28原则场景分流
- 来源：ArkClaw 企业推广策略 deck
- 定义：20% 核心场景有高度，80% 长尾场景有广度
- 适用场景：产品定位、GTM 策略、资源分配
- 使用方式：先识别核心场景（高价值高壁垒），再覆盖长尾

### 双维度场景评估
- 来源：ArkClaw 企业推广策略 deck
- 定义：技术可行性 × 商业 ROI 双维度筛选
- 适用场景：竞品分析、产品路线图规划
- 使用方式：四象限矩阵，优先右上角（高技术可行 + 高 ROI）

### 系统化创作 vs 碎片化创作
- 来源：dontbesilent Claude Code 工作流调研
- 定义：碎片化 = 每次从零开始；系统化 = 每次给系统添砖加瓦
- 适用场景：内容生产流程设计、AI 工作流设计
- 使用方式：建立素材库 → 创作前检索 → 创作后沉淀 → 数据反哺方法论

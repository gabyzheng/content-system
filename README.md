# Content System

系统化内容生产数据目录。

> 📦 规范定义在 Skill：`content-pipeline`（`~/.openclaw/workspace/skills/content-pipeline/SKILL.md`）
> 触发词：选题跟进、发公众号、生成日报、写竞品分析、做PPT、做H5、飞书风格PPT、HTML版PPT、h5 deck、出卡片、质量检查

## 目录

```
01-选题管理/       # 选题收集箱，00-选题记录.md 追踪所有选题状态
02-选题内容/       # 选题工作目录（按类型 → 序号-选题名称 → 阶段文件夹）
  ├── 公众号文章/
  ├── PPT/
  ├── H5-Deck/     # 飞书风格 HTML 幻灯片（feishu-deck-h5 技能）
  ├── 日报/
  ├── 竞品分析/
  └── 社交卡片/
03-素材库/         # 可复用素材（核心概念、金句、数据、案例）
04-方法论沉淀/     # 方法论 + 质量检查清单模板
```

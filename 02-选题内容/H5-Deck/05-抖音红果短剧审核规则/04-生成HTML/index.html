抖音专项
DOUYIN ONLY

抖音特殊规则 — 34条红线 + 四大高风险场景

四大高风险场景（90%拒审来源）
  • 关系越界：将控制/羞辱/物化包装为"甜宠"，不伦擦边关系刻意制造讨论 → 直接高风险
  • 感官刺激引流：露骨/惊悚开篇+封面标题暗示 → 即使后续合规仍整体下架
  • 失真逻辑洗白：为作恶角色设无代价获利，用架空设定美化违规违法 → 违背常识即下架
  • 低质拼接冒充：PPT轮播+伪动态+机械朗读+AI穿帮 → 基本无法过审
AI标识（强制 · 错即下架无修改机会）
  • AI生成内容≥50%：片头前3秒 + 封面 + 简介 三处标注"本作品含AI生成内容"
  • 未标注或标注不显著 → 直接限流/下架
  • 主动标注不影响流量；被平台检测补标"疑似AI生成"影响权重
制作质量硬指标
  • 禁止纯静态PPT拼接 / 伪动态凑数
  • 单集时长建议≥30秒（拒绝几秒一集"水文"）
  • 禁止全程机械配音 / 音画不同步
  • 禁止人物穿模 / 马赛克 / 烂尾断更 / 水印外链
  • AI技术瑕疵率>5% 即限流
  • 封面禁用"热播""爆款""强推"等营销词
  • 封面禁止视觉擦边或主体五官缺失
剧本限投 & 审核机制
  • 单个账号30天内签约剧本 ≤10本
  • 实名制绑定、禁止代投
  • 分类分层审核 + 分级备案制度
  • 审核覆盖剧名/封面/简介/正片/配音/字幕/画质全链路
红果专项
HONGGUO ONLY

红果特殊规则 — 3类禁拍 + 六大通用红线 + 真人专项加码

3类直接禁拍（零容忍·永久禁止）
  • 灵异迷信：鬼魂通灵、算命驱邪等全面封禁；诡异须用人为布局/心理错觉解释
  • 宗教民俗：宗教邪术永久禁止；不得解构宗教教义、编造法术修炼；民俗仅可生活化纪实
  • 缅北猎奇：囚禁/虐待/器官买卖不准拍摄；反诈仅保留公益科普形式
六大通用红线（4月落地）
  • 超自然金手指：隐身/瞬移/读心/快速自愈/修仙炼气/上古传承/凭空造物→全面禁拍。能力主角仅限退役军人/运动员/侦察兵等现实背景
  • 以暴制暴：主角不得主动伤人/私刑囚禁/聚众围堵；反派交公安处置；主角仅能躲闪格挡
  • 危险道具&血腥：匕首/弹簧刀/毒针迷药禁出镜；刀具抵脖/出血特写禁拍；冲突仅用木棍等轻微磕碰，全程无受伤画面
（续）六大通用红线
  • 人格羞辱&极端价值观：扔钱辱人/逼迫下跪/掌掴→叫停；禁刻意制造贫富对立；禁校园职场霸凌
  • 隐私侵犯&要挟：禁偷拍窃听/私自调监控/曝光隐私；禁用弱势亲属胁迫主角；禁威胁医院停药
  • 交通暴力&恶意毁物：禁恶意别车/危险飙车/蓄意撞击；砸车窗/踹车身/车内打斗→删减
真人专项加码（5月启动）
  • 男频保镖/豪门：无资质私人保镖/黑市佣兵→立项驳回；雇凶暗杀/灭口→大幅修改
  • 婆媳/婚恋收紧：霸总禁锢/绑架追求/PUA精神控制/婚内出轨/小三上位/家暴流产→全部禁止
  • 价值观硬控：无脑砸钱炫富/拜金主义/权贵凌驾法律→禁止；反派须移交司法，完整呈现抓捕判刑
  • 标题/剧名禁用词："灭门""逆天改命""一手遮天"等博眼球词→禁用
  • 经典案例：《隐身侍卫》因违规删减，流量大幅下滑

📋 内容红线检查
  • 🔴 主角是否有尊严（不是工具/符号/笑料）？
  • 🔴 关系是否有边界（不靠越界身份制造卖点）？
  • 🔴 爽点是否靠价值观扭曲（不靠拜金/复仇/阶级对立）？
  • 🔴 超自然设定是否用科学逻辑收尾？
  • 🔴 封面标题是否低俗擦边/博眼球？
  • 🔴 有没有AI换脸/盗用素材/洗稿？
  • 🔴 反派最终是否移交司法机关？
✅ 前置合规 & 长期策略
  • 📝 备案先行：开拍前确认备案类型（一类/二类/三类）
  • 📝 AI素材溯源：所有AI生成素材保留生成记录
  • 📝 肖像授权：使用任何真人形象须有书面授权
  • 📝 AI标注到位：≥50%AI内容三处标注（片头+封面+简介）
  • 💡 从"爽点模板"转向原创叙事

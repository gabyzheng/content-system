# PPT 检查清单

> 发布前逐项核对，全部通过方可发布。

## 内容检查

- [ ] 观点已打透（不是信息罗列，有明确主张）
- [ ] 每页一个核心观点
- [ ] 逻辑线清晰（问题 → 分析 → 方案 → 行动）
- [ ] 数据有来源标注

## 设计检查

- [ ] 风格：蓝色简约咨询风
- [ ] 比例：16:9
- [ ] 分辨率：4K
- [ ] 有适当留白，不拥挤
- [ ] 插图服务于观点，非装饰

## JSON 检查

- [ ] 总览 JSON 完整（快速预览全篇）
- [ ] 逐页 JSON 完整（高清逐页生成）
- [ ] JSON 结构符合 ppt-json-workflow.md 规范

## 质量检查

- [ ] 无错别字
- [ ] 无 AI 幻觉内容
- [ ] 标题简洁有力
- [ ] 正文信息密度适中
